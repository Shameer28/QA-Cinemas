# QA Cinema Back end
This is the api for qa cinemas

Prerequisites
```
NPM
Mailserver
```
How to install
```
run npm install
```
This will install dependancies

Then create a secret.js and fill it with this information
```
{
	"pv_stripe_key":"stripe_key",
	"email": {
		"user": "emailOfWebserver",
		"pass": "pass"
	},
	"contactEmail": "emailToSend" 
}
```
